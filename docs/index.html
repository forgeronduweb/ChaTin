Comment installer un APK ?
  1. Télécharge le fichier depuis le bouton ci-dessus.
  2. Ouvre-le depuis tes notifications ou ton dossier "Téléchargements".
  3. Si Android te demande d'autoriser "Sources inconnues", accepte — c'est normal pour une app installée hors Play Store.
  4. Installe, puis lance ChaTin 🎉
